5. With mock data (averageScore=7.0), "Room for Improvement" alert should NOT be visible. Verify `cy.contains('Room for Improvement').should('not.exist')`.

6. Trigger the alert: use Cypress to change bedroom sliders to low values. The RatingField uses a slider with `aria-valuetext`. **Important notes for slider interaction:**
- RatingField likely uses a shadcn Slider component. To change the value, use `cy.get('input[type="range"]')` or find the slider by label and use `.invoke('val', 2).trigger('input')`. Alternatively, use `cy.contains(label).closest('[data-slot="form-item"]').find('[role="slider"]')` and trigger value change via keyboard: `.type('{leftarrow}{leftarrow}{leftarrow}{leftarrow}{leftarrow}')` to decrease from default values.
- If slider interaction is problematic, an alternative is to find the slider thumb and use `.trigger('mousedown').trigger('mousemove', { clientX: ... }).trigger('mouseup')`. The simplest reliable approach is keyboard arrows on the focused slider.
